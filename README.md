# 🚀 AI Social Media Sentiment Intelligence Dashboard

A premium Machine Learning + NLP dashboard that analyzes social media comments/posts and classifies sentiment into:

✅ Positive  
✅ Negative  
✅ Neutral  

Built for portfolio, placements, internships, and GitHub proof of work.

---

## 📌 Features

- Premium Black Theme Dashboard
- Real-Time Sentiment Prediction
- KPI Cards
- Pie Chart Analytics
- Weekly Trend Graph
- Word Cloud Visualization
- CSV Download Report
- Streamlit Interactive UI

---

## 🧠 Tech Stack

- Python
- Pandas
- Streamlit
- Plotly
- Matplotlib
- WordCloud
- VS Code
- GitHub
